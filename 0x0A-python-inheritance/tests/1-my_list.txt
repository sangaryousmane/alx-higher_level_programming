Edge cases for the MyList class
======================================

The MyList inherits from the list class and sorts the list in ascending order

USAGE
=======

Imports the module for processing
>>> MyList = __import__('1-my_list').MyList

Apply the sorted function after appending integers
>>> my_list = MyList()
>>> my_list.append(3)
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(1)
>>> my_list.append(5)
>>> my_list.append(9)
>>> my_list.append(2)
>>> my_list.append(6)
>>> my_list.append(5)
>>> my_list.append(3)
>>> my_list.append(5)
>>> my_list.print_sorted()
[1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]

Append 7 and resort to see the effect of the print_sorted function
>>> my_list = MyList()
>>> my_list.append(7)
>>> my_list.append(2)
>>> my_list.append(1)
>>> my_list.print_sorted()
[1, 2, 7]
            
Remove a value and see the effect
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(10)
>>> my_list.append(2)
>>> my_list.remove(1)
>>> my_list.print_sorted()
[1, 2]
            

Clear all the items from the list
>>> my_list = MyList()
>>> my_list.append(10)
>>> my_list.clear()
>>> my_list.print_sorted()
[]
